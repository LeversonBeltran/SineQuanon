Para crear un nuevo proyecto en Git Hub,
se debe seguir los siguientes pasos:

1. Se crea el repositorio directamente sobre la carpeta del proyecto
para ello se usa la consola Git Bash para iniciar el proyecto
sobre la ruta o carpetas. Se inicia con el comando
git init

2. Luego se valida con el comando:
git status

3. Se agregan las carpetas o archivos con el comando:
git add --all

4. Se inserta el primer Commit:
git commit -m "Descripción del ajuste o modificación"
